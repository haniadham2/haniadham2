## How I Think About AI Products

```
User Problem  ──►  Hypothesis  ──►  System Design  ──►  Evaluation  ──►  Iteration
```

Good AI products don't start with models. They start with a user who has a real problem,
a hypothesis about how AI solves it better than anything else,
and a clear definition of what "working" looks like before a single line of code is written.
